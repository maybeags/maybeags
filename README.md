## 🚀 주요 프로젝트

### 🚗 Car Database 풀스택 (Spring Boot + React + AWS)
- Spring Boot 3.x + JPA + Spring Security + JWT + Swagger
- Car/Owner 1:N 엔티티 설계, BCrypt 암호화, CORS, Role 기반 접근제어
- React(TypeScript) + MUI + AG-Grid + react-query 프론트엔드
- AWS EC2 배포, Nginx 리버스 프록시, Certbot HTTPS, RDS/S3 연동

### 🔐 Google OAuth2 소셜 로그인 풀스택
- Spring Boot OAuth2 Client + JWT 통합 인증 구현
- React(TS) + MUI + PrivateRoute/GuestRoute 라우팅 보호
- Google 인증 콜백, 쿼리스트링 토큰 전달 처리
- 👉 [korit_12_oauth2](https://github.com/maybeags/korit_12_oauth2)

### 🤖 Gemini AI Chatbot
- JavaScript ES6+ + Fetch API + Google Gemini API 연동
